Refresh for 10 New Jokes!


About This API

I found this API through GitHub pages. The feature of the API is really interesting, as each time you refresh the page, the jokes change. Every refresh gives the readers 10 new jokes randomly selected from the database. To retrieve the data, I used forEach() since there are 10 Jokes in the doc, and I retrieved by their setups and their punchlines. The request for API, I simply used GET.

Two Potential Applications

1)If you can't think of any new jokes on your own, of if you only have one or two jokes in your pocket, this API is really helpful for you to entertain your friends through new jokes.

2)Just like how Netflix distributes content using API, if an application is collecting and distributing joke contents to users, it's helpful for them to have access to this API and other APIs that contains lots of data on jokes.
